# AdventOfCode2025 🎄

Solutions to [Advent of Code 2025](https://adventofcode.com/2025/), implemented in Swift.

Each day is a fully standalone executable target.  
This makes every solution isolated, clean, and easy to run or test independently.

## Build & Run 🚀

Build all targets:

```bash
swift build
```

## Run a specific day 🐝

```bash
swift run day01
````
